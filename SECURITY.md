# Security

This repository contains an agent skill with instructions and markdown references. It does not require secrets, API keys, or elevated local permissions.

## Report A Security Issue

Open a private security advisory on GitHub if available, or open an issue without including sensitive details.

## User Guidance

Always review skills before installing them. This skill may instruct an agent to inspect UI files and run local development servers for screenshot verification. It should not ask the agent to copy credentials, exfiltrate private data, or publish code without explicit user approval.

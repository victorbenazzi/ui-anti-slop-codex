# skills.sh Publishing Notes

English first. Portuguese follows.

## English

skills.sh indexes installable GitHub skills through the `skills` CLI. A skill can appear after users install it with:

```bash
npx skills add victorbenazzi/ui-anti-slop-codex
```

This repository includes:

1. `SKILL.md` in the repository root.
2. `references/` with detailed guidance loaded by the skill.
3. `agents/openai.yaml` with UI metadata.
4. `skills.sh.json` to customize the repository page when skills.sh sees the repo.
5. README and docs in English with Portuguese secondary documentation.

The expected skills.sh page is:

```text
https://skills.sh/victorbenazzi/ui-anti-slop-codex
```

If it does not appear immediately, install once with the `skills` CLI and wait for the cache to refresh.
